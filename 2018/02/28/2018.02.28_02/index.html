新年开启新篇章

发表于 2018-02-28

回顾

元旦前后，每天都会看到大神博主们的年终总结，看着他们的总结，突然想起我这很久没有更新的博客。从17年6月到9月，陆续写了几篇文章，十一过后就渐渐搁下了。虽说不时会在本地记录下平时遇到的问题和发现的解决方法，但是却一直没有更新在博客上。我的年终总结写在今天，因为在我看来，农历年过了才是真的过完了一年（算不算是我偷懒的借口QAQ~）

回想起来，发现今年的假期过得特别的快。可能是由于去年我还是一个学生，有着一个多月的假期，今年就突然成了按照法定假日放假的“打工仔”了。七天的假期和之前相比一下让我感到了落差（从之前的第一个到家最后一个离家到现在最后一个到家第一个离家）。不过我还是有一股莫名的干劲的，觉得工作使人快乐QAQ~

现在

十一之后从天津转战上海，找到了上海的第一份工作。公司规模不大，但氛围让我感觉很舒服。进入公司刚好就能接手到刚刚进行的项目，这可能是一件比较幸福的事情了。从一开始着手开始做项目到现在，基本上项目的每一个模块我都有过参与，每次Bug修复时能很快的定位到错误，还是让我小有成就感的。

但同时也让我学到了很多，看着比我先从事工作的人，他们对于代码的处理逻辑，对于问题的处理态度，让我受益良多。开始很多时候，我都是靠着代码写的次数多了，才比较熟练掌握，遇到问题总是先考虑怎么避过或处理。但是他们不一样，很多时候他们都是定位错误的发生位置和原因。学习之路还很长，最近我已经深深意识到，要加快学习的步伐，不然，Android9都要出了，我却还在6.0的时代。

可能由于个人性格，我想，慢慢的就会融入氛围当中，成为营造气氛的一份子，这种事我还是比较倾向于顺其自然。去年比较揪心的就是考驾照了，当初在学校，周围一群人问我要不要考驾照，我都坚决的说，不着急。结果，刚毕业的那段时间我却抽了风似的报考了驾校。问题还是在天津报的…导致现在还得请假到天津考试，幸亏之前已经考完了科二，科三也练了几次（虽说挂了一次），剩下的科四，也就这段时间就能结束了，驾照已经拿到一大半了！

马上开始的未来

今年，一定不能再把博客给放下了，经常更新博客是我今年的一大目标。现在每次写博客的时候，总是不知道怎么能让文章读起来更加易懂流畅，希望可以越来越好。然后慢慢学习新的知识，重要的是不同时进行多项新知识的学习，感觉以前这样效率很低。定好学习的目标，不管好坏，至少应该学习到了解掌握，能够应用，不然每次断断续续什么都学不好，这点很重要！

希望今年可以多看看书，不管是技术类还是其他。我觉得老大说的很对，每天只要读10页，一本五六百页的书也就两个月看完了。现在想来，我完整看完的书貌似屈指可数（除了学校的课本…）。我觉得如果这件事坚持了下来，其收益不比写博客来得少。很多事都是贵在坚持，虽然每天做的很少，但是如果持续下去，那么收获肯定是巨大的。

最后，希望家人一切都好，顺顺利利，健健康康。也希望大家新的一年越来越好…
